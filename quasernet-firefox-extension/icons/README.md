# Icônes de l'extension

Pour créer les icônes de l'extension, vous pouvez :

1. **Utiliser un outil en ligne** comme :
   - https://realfavicongenerator.net/
   - https://www.favicon-generator.org/

2. **Créer manuellement** des fichiers PNG aux tailles suivantes :
   - icon16.png (16x16)
   - icon32.png (32x32)
   - icon48.png (48x48)
   - icon128.png (128x128)

3. **Utiliser le logo existant** :
   Vous pouvez redimensionner le logo de Quasernet situé dans `../journal-culturel-v8-HTML-améliorationCSS/logo/`

## Spécifications

- Format : PNG avec transparence
- Fond : Transparent ou couleur unie
- Style : Minimaliste et reconnaissable
- Couleurs : Cohérentes avec la charte graphique Quasernet

## Icônes temporaires

En attendant les icônes finales, vous pouvez utiliser des placeholders SVG convertis en PNG.
